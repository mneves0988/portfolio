# Portfólio de Automação e IA

Este é o código-fonte do meu portfólio profissional, construído com React, TypeScript, Vite e Tailwind CSS.

## Tecnologias

- **Vite**: Build tool rápida e moderna.
- **React**: Biblioteca para construção de interfaces.
- **TypeScript**: Superset de JavaScript com tipagem estática.
- **Tailwind CSS**: Framework de CSS utilitário para estilização rápida.
- **Shadcn UI**: Componentes de interface reutilizáveis e acessíveis.

## Como Executar Localmente

1. Clone o repositório:
```bash
git clone <URL_DO_SEU_REPO>
cd header-harmony-portfolio
```

2. Instale as dependências:
```bash
npm install
```

3. Inicie o servidor de desenvolvimento:
```bash
npm run dev
```

## Publicação

Este projeto está configurado para ser publicado no GitHub Pages.
Para realizar o build de produção:

```bash
npm run build
```

Os arquivos estáticos serão gerados na pasta `dist`.
